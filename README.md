# live-cursors

real-time collaborative cursor & drawing playground. see where others are pointing & draw together in shared rooms.

## features
- live cursor tracking across users
- collaborative drawing
- room-based isolation
- color-coded cursors & names

## setup
```bash
npm install
npm start
```

runs on `http://localhost:3000`. visit `/room/your-room-name` to create or join a room.

## tech
websockets & vanilla js. no frameworks, no nonsense.
